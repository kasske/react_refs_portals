# React js refs and portals

React js mini project messing with refs and portal.
Used vite to create the app

## Concepts

- refs
- forwarding refs
- useImperativeHandle Hook
- multiple slots


## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
